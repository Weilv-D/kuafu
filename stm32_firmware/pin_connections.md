# 夸父 KuaFu — STM32F407ZGT6 全系统针脚连接方案

> 版本：v1.0 / 2026-07-09
> 主控芯片：STM32F407ZGT6（LQFP144，实物丝印省略"P"，如 PB8 标为 B8）

---

## 一、串口资源分配总览

| 串口外设 | 功能用途 | STM32 引脚 (丝印) | 波特率 |
| :--- | :--- | :--- | :--- |
| **USART1** | CH340 串口下载 / PC 调试打印 | PA9 (TXD) / PA10 (RXD) | 115200 bps |
| **USART2** | RS485 模块 → DDSM315 电机总线 | A2 (TX) / A3 (RX) | 115200 bps |
| **USART3** | 舵机控制板 → ST3215 总线 | B10 (TX) / B11 (RX) | 1000000 bps |
| **USART6** | 树莓派 5 主控桥接串口 | C6 (TX) / C7 (RX) | 921600 bps |
| **I2C1** | BMI088 IMU 姿态传感器 | B8 (SCL) / B9 (SDA) | 400 kHz |

## 二、全系统针脚连接大表

### 1. CH340 串口下载器（程序烧录 + PC 调试）

| CH340 模块引脚 | 信号方向 | STM32 引脚 | 实物丝印 | 说明 |
| :--- | :---: | :--- | :---: | :--- |
| 3.3V | ──► | 3.3V 供电 | **3V3** | 临时供电；若已接电池则**不接此线** |
| TXD | ──► | PA10 (USART1_RX) | **RXD** | 电脑发送 → 单片机接收 (接板载 RXD) |
| RXD | ◄── | PA9 (USART1_TX) | **TXD** | 单片机发送 → 电脑接收 (接板载 TXD) |
| GND | ─── | GND | **GND** | 必须共地 |
| rst | ✗ | 悬空不接 | — | CH340 的 RST 不能复位 STM32，不接 |

**⚠️ 下载操作步骤：**
1. 将板子上的 BOOT0 跳线帽拨到 **1（接 3.3V）**
2. 打开 FlyMcu 或 STM32CubeProgrammer，选择对应 COM 口，载入 `.hex` 文件
3. 手动按板子上的 **RST 复位键** 启动下载
4. 下载完毕后将 BOOT0 跳线帽拨回 **0（接 GND）**，按复位键运行程序

---

### 2. 树莓派 5 ↔ STM32（主控桥接通信）

| 树莓派 5 专用 UART 口 (JST-SH 1.0mm) | 信号方向 | STM32 引脚 | 实物丝印 | 说明 |
| :--- | :---: | :--- | :---: | :--- |
| Pin 1 (TX) | ──► | PC7 (USART6_RX) | **C7** | 树莓派发送 → STM32 接收 |
| Pin 2 (GND) | ─── | GND | **GND** | 信号共地 |
| Pin 3 (RX) | ◄── | PC6 (USART6_TX) | **C6** | STM32 发送 → 树莓派接收 |

- 电平：3.3V TTL，两端直连无需分压电阻
- 波特率：921600 bps
- 树莓派设备节点：`/dev/ttyAMA0`（需关闭系统控制台占用）

---

### 3. 舵机控制板 ↔ STM32（ST3215 四髋舵机驱动）

| 舵机控制板引脚 | 信号方向 | STM32 引脚 | 实物丝印 | 说明 |
| :--- | :---: | :--- | :---: | :--- |
| RXD | ──► | PB10 (USART3_TX) | **B10** | STM32 发送 → 控制板接收 |
| TXD | ◄── | PB11 (USART3_RX) | **B11** | 控制板发送 → STM32 接收 |
| GND | ─── | GND | **GND** | 信号共地 |

- 电平：3.3V TTL
- 波特率：1 Mbps
- 控制板逻辑端自供电，**不接 STM32 的 3.3V**
- 舵机动力端（12V）单独由下甲板 Buck 降压模块供电
- 4 台 ST3215 舵机菊花链串联，ID 分配：LF=1，RF=2，LB=3，RB=4

---

### 4. RS485 转换模块 ↔ STM32（DDSM315 双轮毂电机）

| RS485 模块引脚 | 信号方向 | STM32 引脚 | 实物丝印 | 说明 |
| :--- | :---: | :--- | :---: | :--- |
| RX | ──► | PA2 (USART2_TX) | **A2** | STM32 发送 → 模块接收 |
| TX | ◄── | PA3 (USART2_RX) | **A3** | 模块发送 → STM32 接收 |
| V (VCC) | ──► | 3.3V | **3V3** | 模块逻辑供电 |
| GND | ─── | GND | **GND** | 信号共地 |

- RS485 模块需为**带自动收发流控**版本（无需额外 DE/RE 方向控制引脚）
- 波特率：115200 bps（DDSM315 固定波特率）
- RS485 差分总线侧（485_A / 485_B）并联接两台电机
- 电机 ID 分配：左轮=1，右轮=2
- 软件采用分时轮询：Slot0 轮询 ID1（左），Slot1 轮询 ID2（右），周期 4ms (250Hz)

---

### 5. BMI088 IMU ↔ STM32（I2C 模式，本体姿态感知）

| BMI088 模块引脚 | 信号方向 | STM32 引脚 | 实物丝印 | 说明 |
| :--- | :---: | :--- | :---: | :--- |
| VCC | ──► | 3.3V | **3V3** | IMU 供电 |
| GND | ─── | GND | **GND** | 信号共地 |
| SCL | ──► | PB8 (I2C1_SCL) | **B8** | I2C 时钟线 |
| SDA | ◄──► | PB9 (I2C1_SDA) | **B9** | I2C 数据线（双向） |
| INT3 | ◄── | PB1 (EXTI1) | **B1** | 陀螺仪数据就绪中断，用于 1kHz 同步触发 |
| INT1 | ✗ | 悬空不接 | — | 加速度计中断，暂不使用 |
| INT2 | ✗ | 悬空不接 | — | 加速度计中断，暂不使用 |
| INT4 | ✗ | 悬空不接 | — | 陀螺仪中断，暂不使用 |

- I2C 总线速率：400 kHz (Fast Mode)
- 加速度计 I2C 地址：`0x18`，陀螺仪 I2C 地址：`0x68`
- SCL/SDA 线上需有 **4.7kΩ 上拉电阻**（通常已集成在 BMI088 模块板上）
- SCL/SDA 导线总长度 **< 15 cm**，远离强电干扰线

## 三、布线规范与注意事项

| 规范 | 要求 |
| :--- | :--- |
| **强弱电隔离** | 动力电（18.5V/12V 大电流）与信号线（UART/I2C）物理间距 ≥ 2 cm |
| **I2C 线长** | SCL/SDA 总长 < 15 cm，推荐 < 10 cm |
| **UART 线长** | 各 UART 线长 < 30 cm（1 Mbps 舵机线 < 20 cm） |
| **单点共地** | 所有模块 GND 汇集到分电板主 GND 焊盘（星型接地） |
| **双路供电禁止** | CH340 的 3.3V 和电池不可同时向 STM32 供电 |
| **串口编程模式** | 下载时 BOOT0 拨 1；运行时 BOOT0 拨 0 |
