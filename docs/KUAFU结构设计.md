# 夸父 KUAFU — 结构设计

设计以 DDSM315 与 ST3215 C018 的性能与尺寸为硬约束。机构四参数（d / a / b / D₀）经蒙特卡洛采样 10 万配置、Pareto 前沿提取、200 组 Dirichlet 分布权重稳健性扫掠验证；选定解 (d=52, a=93, b=149) 在工程先验权重（扭矩/行程主导，传动角/条件数次要）下综合胜出率 50.5%、居候选第一，扭矩峰值 1.92 Nm、驻留扭矩 0.51 Nm、传动角全程 ≥31°、工作空间行程 149 mm 均满足硬指标。半轮距 Y（=98 mm）作为整机稳定性-宽度折中的独立决策（不参与机构优化，见 §二）。驻留态为最低姿态（曲柄贴合机身，D₀=58），舵机近零力矩。所有尺寸经计算加裕度确定。坐标系：原点在底盘几何中心，+X 前 / +Y 左 / +Z 上，Z=0 地面。

> 优化脚本见 `analysis/optimize/`（monte_carlo.py / dirichlet_weights.py / plot_optimization.py），可经 `python run_all.py` 复现，结果图 `output/pareto_front.png`、`output/dirichlet_winrate.png`。

## 四、整机布局

双层盒式框架：两侧板（240×55×4，Y=±98）+ 上下甲板。电池下沉到两轮间凹槽以最小化 COM。

机身 X 向长度 240 mm 由连杆扫掠空间决定。驻留态 D₀=58 时 P1 位于 X≈−107，连杆向内穿行需要机身半长 ≥107 mm，取 120 mm（含 13 mm 安全间隙）得整长 240 mm。半轮距 Y=98，侧板间距 196 mm。机身底 Z=97 mm。

## 五、腿部设计

对称并联五杆机构，每腿 2-DOF 连续可控。

```
     髋面 ●A ─── d=52 ───●B          ← A(−26,0) B(26,0)
         │                 │
    a=93 │            a=93 │           ← 对称曲柄
         │ \             / │
         │  \P1       P2/  │            ← P₁ 落 Q 左、P₂ 落 Q 右
   b=149 │   \         /  │ b=149       ← 对称连杆
         │    \       /   │
         │     ●Q(0)      │            ← 轮毂中心，向 +X 滚
```

对称非交叉构型：P₁.x<0、P₂.x>0，连杆平行不交叉。左右腿零件完全通用，杆长种类仅 2 种。运动学可达 D₀ 58–207 mm，补偿带 58–110，爬阶高度 149 mm。

### 5.1 两态承力模型

| 状态 | 曲柄位姿 | 承力路径 | 舵机受力 |
|-----|---------|---------|---------|
| **驻留态**（轮行 / 站立，D₀=58） | 贴合承重面 | 轮→连杆→铰链→曲柄杆身→承重面→侧板→机架（绕过舵机轴） | **≈0**（仅保持电流 ~50–80 mA/台） |
| **瞬态**（补偿 / 爬阶，D₀>58） | 离开承重面 | 轮→连杆→铰链→曲柄→舵机轴 | 0.51–1.92 Nm（§5.4） |

承重面为侧板内侧匹配 D₀=58 时曲柄角度的斜凸台。D₀=58 时曲柄指向 α₁≈−209°（约水平上方 29°），凸台角度与之匹配。因构型完全对称，左右承重面几何一致。

载荷自紧负反馈：载荷经连杆产生拉力，其分量将曲柄压向承重面，越载越紧，为纯被动物理自锁，无需主动控制维持。瞬态动作结束后必须立即回 D₀=58 使曲柄重新贴合锁止，不可长时悬空。

### 5.2 腿杆强度校核

杆件材料为 **PETG**（承力件，抗摔韧性与耐热优于 PLA；侧板/甲板等非承力件仍用 PLA），深边立放于 X-Z 面内。截面：大腿 13(深)×10(厚) mm，小腿 16(深)×10(厚) mm。轴向载荷为主，弯曲挠度非控制因素。

**轴向应力**：以堵转工况估算，设计载 30 N/腿。大腿截面 130 mm²，σ≈0.23 MPa；小腿截面 160 mm²，σ≈0.19 MPa。均远低于 PETG 拉伸强度 50 MPa。

**欧拉屈曲**（弱轴，PETG E=2000 MPa，K=1）：大腿 I=1083 mm⁴，P_cr≈2472 N，裕度 82×；小腿 I=1333 mm⁴，P_cr≈1185 N，裕度 40×。均远超工程安全线 4×。

**面外（Y 向）侧弯**：转向或侧倾时连杆承受 Y 向分量。取最坏情况 F_y = 30×20% = 6 N 作用于杆端。大腿面外 I_y=1831 mm⁴，σ≈1.98 MPa，挠度 ~1.3 mm；小腿 I_y=3413 mm⁴，σ≈2.10 MPa，挠度 ~3.5 mm。均远低于 PETG 弯曲强度 75 MPa。正常工况下 Y 向力来自轮胎回正力矩，远小于 6 N。

**面内弯曲挠度**（全侧向载荷于杆端，仅参考）：大腿 ~2.1 mm，小腿 ~7.1 mm。正常轴向受力时不达此值。

### 5.3 接口

**舵机→侧板**：4×M3（Ø3.3 通孔）@ r14 + Ø20.5 中孔过舵盘 Ø19.95。舵盘→大腿以 M3 螺栓紧固。

**转动副**（A/B/P1/P2/Q，5 处/腿）：打样阶段 M3×20 螺栓 + 平垫 + 尼龙防松母直穿 PLA Ø3.2 孔。孔磨旷后升级为 MR63ZZ（3×6×2.5 mm）微型轴承。升级方法：预留 Ø6 导孔用于手工铰孔对中，铰至 Ø6 后压入轴承；若孔超差则以环氧树脂填孔后重新钻铰。

**Q 处错层**：两连杆 Y 向垫片错层 4 mm，互不摩擦。

**轮端叉口**：4×M2.1 @ Ø15.2 接电机面，内宽 39 mm，Ø84 让位。M2.1 螺纹在 PLA 中易滑牙，叉口螺孔处预埋 M2 热熔铜螺母（外径 Ø3.5，加热嵌入）或改 M2 螺栓 + 防松母对穿。

**关节动力学特性（面向 RL 建模与控制补偿）**：被动铰链为无阻尼刚性转动副，无润滑。打样阶段（M3 螺栓直穿 PLA）等效干摩擦 μ≈0.3；升级 MR63ZZ 后为滚动摩擦 μ≈0.001。新机转动副间隙 ≤0.3 mm（§九验收），干摩擦下渐进磨损。五杆腿为多刚体摆系统，一阶腿摆频率估算 1.5–2 Hz，结构阻尼比 <0.05，扰动后振荡衰减慢。控制层需在平衡环加入腿关节角速度的陷波/低通滤波（notch 陷在实测腿摆频率），避免控制器响应腿自振；驻留态舵机宜输出小扭矩预紧压向承重面，用电磁力矩充当阻尼抵消脉冲冲击下的离面趋势。

### 5.4 五杆静力学（瞬态工况）

以下适用于曲柄离开承重面的瞬态。驻留态载荷绕过舵机，扭矩 ≈0。

模型：A(−26,0)−B(26,0)，d=52；a=93，b=149；Q=(0,−D₀)。载荷 F=30 N/腿（含 3× 冲击），静力平衡法求解。对称构型下 τ₁=τ₂。

| D₀ | τ₁=τ₂ (Nm) | γ (°) | 占连续安全 1 Nm |
|---|---|---|---|
| 58（驻留，悬空参考） | 0.51 | 46 | 51% |
| 70 | 0.91 | 51 | 91% |
| 85 | 1.30 | 53 | 130% |
| 100 | 1.58 | 53 | 158% |
| 115 | 1.77 | 51 | 177% |
| 145（峰值） | 1.92 | 47 | 192% |
| 175 | 1.83 | 40 | 183% |
| 207（上限） | 1.44 | 31 | 144% |

瞬态扭矩范围 0.51–1.92 Nm。D₀≤70 在连续安全值内；D₀>70 超连续安全但低于堵转 2.94 Nm，属短时脉冲（补偿毫秒级、爬阶秒级），舵机可承受。传递角全程 ≥31°，力传递合格。脉冲动作后须回承重面锁止。

舵机热安全性：最坏工况 D₀=145 时单次脉冲发热约 12.5 W×1 s，铝壳舵机（~60 g）温升仅 ~0.17 K/脉冲，裕度极大。频繁连续脉冲（≥0.5 Hz）由首件实测确认。
