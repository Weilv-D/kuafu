# 夸父 KUAFU — 桌面双轮腿机器人

面向强化学习的桌面级双轮腿验证平台。"大脑 + 小脑"分层架构，整合轮式平衡、腿式姿态变换与视觉导航闭环。

坐标系：原点在底盘几何中心（髋面中心，两髋点 A-B 中点），+X 前 / +Y 左 / +Z 上，Z=0 地面，轮恒接地 Z=r，机身底 Z=D₀+r。符号定义见[附录 A](#附录-a符号与坐标系)。

| 约束 | 目标值 |
|------|--------|
| 整机重量 | ≤ 2.7 kg（核算 **2.68 kg**，含充电宝） |
| 底盘宽 | ~231 mm（含胎），下甲板侧壁间距 196 mm |

| 控制周期 | 姿态估计 **1 kHz**（IMU）；平衡控制 **~250 Hz**（RS485 电机环）；决策 **30–50 Hz**（Pi） |

### 任务范围与验收标准

| 任务 | 量化指标 | 关键依赖 |
|------|---------|--------|
| **轮行平衡**（平地） | 连续平衡 ≥10 min，3 次侧向轻推（~2 N·s）恢复时间 ≤2 s | STM32 平衡环 + 扭矩裕度 |
| **腿姿态变换** | 左右地形差 ±25 mm 内保持机身水平，切换时间 ≤0.5 s，静态误差 ≤2° | 腿垂向行程 + 承重面自锁 |

机构物理补偿能力在驻留态可达 ±45 mm，中段（D₀=100–145）峰值 ±118 mm，对 ±25 mm 需求有 4× 以上富余。

## 二、系统架构

### 分层架构

| 层 | 硬件 | 职责 |
|----|------|------|
| 大脑 | 树莓派 5（8GB） | ROS 2 Jazzy、D435i VIO/SLAM、RL 策略推理（ONNX） |
| 小脑 | STM32F407ZGT6 | 姿态估计 1 kHz Mahony；电机环 ~250 Hz；底层平衡控制器 |
| 执行 | DDSM315 ×2 + ST3215 C018 ×4 | 轮毂直驱 + 对称并联五杆髋关节（每腿 2 DOF，连杆不交叉） |
| 感知 | BMI088 + D435i | 本体姿态 + 环境深度 |

控制范式以强化学习为目标，技术路线已锁定为残差 RL（M5）→ Teacher-Student 蒸馏 + RMA（M6），仿真器为 MuJoCo MJX，详见《软件架构与 RL 技术路线设计》。硬件按路线无关的基础设施设计，预留算力与接口兼容所选 RL 范式。

安全关键功能全部下沉 STM32 硬实时层，不依赖树莓派。驻留态零力矩自锁是控制降维的核心机制：D₀=58 时腿成纯被动结构，平衡环只控 2DOF 轮（经典倒立摆），整机降为 2 阶系统、LQR 以 4 维状态 [x,θ,ẋ,θ̇] 求解，RL 状态空间最小，sim-to-real gap 收窄。腿仅在姿态变换和爬阶时参与控制（机制详见[§五 5.1](#51-两态承力模型)）。

### 通信

Pi 与 STM32 之间通过 UART 3.3V TTL 通信，波特率 921600。帧格式：起始 0xA5 + 命令字 1B + 载荷 N B + CRC-8/MAXIM + 结束 0x5A。STM32 上行姿态/状态 ≥200 Hz，Pi 下行控制指令 30–50 Hz。Pi TX→STM32 RX 经 1.8k/3.3k 分压。心跳超时 200 ms 触发 STM32 自主站立保护。

### 供电

**5S LiPo 2200 mAh 95C**：标称 18.5V / 满电 21.0V，~285 g，尺寸 137×43×37 mm。

| 部件 | 规格 |
|------|------|
| 分电板 | 纯铜无源，2 oz |
| 舵机降压 | 同步整流 buck（12V，持续最大 20A）|
| STM32 降压 | MP1584 |
| 树莓派供电 | 充电宝独立 5V/3A+，与主电池仅 GND 单点跨接 |

供电时序：先主电池（STM32 就绪）→ 后充电宝（Pi 启动）；下电先 Pi `sudo shutdown` → 后主电池断开，防止 Pi I/O 向未上电的 STM32 灌电流。

**供电独立性约束（安全关键）**：STM32 必须由主电池经 MP1584 独立供电，**禁止经树莓派 USB 口反向取电**。STM32 板载 USB 口的 VBUS 直连板载 5V 轨，插上即取电——一旦 Pi 崩溃或 USB 总线复位，STM32 会跟随掉电，下方"树莓派崩溃→STM32 独立维持站立"的保护机制即告失效。板载 USB 口仅用于离线烧录/调试，运行时不插。

### 失效模式与保护机制

| 失效场景 | 检测手段 | 保护动作 | 层级 |
|---------|---------|---------|------|
| 舵机过热 | ST3215 内置温度反馈 | 超阈值释放扭矩 → 回承重面锁止 | 固件 |
| LiPo 过放 | 分电板电压采样 | 单节 <3.3V 报警 → <3.0V 断电 | 固件+硬件 |
| 通信中断 | STM32 看门狗 | 平衡环切 PD 兜底，腿回锁，缓减速 | 固件 |
| 树莓派崩溃 | STM32 心跳检测 | STM32 独立维持站立 | 架构 |
| 大倾角摔倒 | IMU 姿态判断 | 舵机卸载 + 电机停转 | 固件 |
| 跌落损伤 | — | PA12 结构 + 轮胎缓冲；关节 M3 螺栓可换 | 结构 |

### 4.3 平衡扭矩

倒立摆模型，M=2.68 kg，r=0.0391 m。每轮准静态保持扭矩 τ = M·g·sinθ·r/2。

| 倾角 θ | 每轮 τ | 占额定 0.55 Nm |
|--------|--------|----------------|
| 10° | 0.089 Nm | 16% |
| 20° | 0.176 Nm | 32% |
| 30° | 0.257 Nm | 47% |

准静态扭矩占额定比例随整机增重而升高（30° 达 47%），直立临界质量裕度收窄至 0.19 kg（2.87−2.68）。大倾角扰动需依赖电机短时过载（堵转 1.1 Nm）。瓶颈仍在 RS485 ~250 Hz 控制环带宽与轮速（额定 200 rpm → 轮缘 0.82 m/s）。

## 五、结构设计

设计以 DDSM315 与 ST3215 C018 的性能与尺寸为硬约束。机构四参数 (d=52, a=93, b=149) 经蒙特卡洛 10 万采样 + Dirichlet 权重扫掠验证，综合胜出率 50.5%。半轮距 Y=98 为稳定性-宽度折中的独立决策，不参与机构优化。

### 5.0 执行器硬约束

执行器规格详见[§三 3.1](#31-执行层)。结构设计的硬约束：DDSM315 轮径 Ø78.16、4×M2.1@Ø15.2 安装；ST3215 堵转 2.94 Nm、连续安全 ~1 Nm、4.7 rad/s、舵盘 Ø19.95 + 4×M3@r14。

### 5.2 机身框架

双层盒式结构：上甲板 + 下甲板。下甲板两侧向下延伸出 Y=±98 侧壁（舵机安装面），维持 196 mm 间距并承载髋关节；电池下沉到两轮间凹槽以最小化 COM。

机身 X 向长度 240 mm 由连杆扫掠空间决定。驻留态 D₀=58 时 P1 位于 X≈−107，连杆向内穿行需要机身半长 ≥107 mm，取 120 mm（含 13 mm 安全间隙）得整长 240 mm。半轮距 Y=98，侧壁间距 196 mm。机身底 Z=97 mm。

### 5.3 腿部设计

对称并联五杆机构，每腿 2-DOF 连续可控。

```
     髋面 ●A ─── d=52 ───●B          ← A(−26,0) B(26,0)
         │                 │
    a=93 │            a=93 │           ← 对称曲柄
         │ \             / │
         │  \P1       P2/  │            ← P₁ 落 Q 左、P₂ 落 Q 右
   b=149 │   \         /  │ b=149       ← 对称连杆
         │    \       /   │
         │     ●Q(0)      │            ← 轮毂中心，向 +X 滚
```

对称非交叉构型：P₁.x<0、P₂.x>0，连杆平行不交叉。左右腿零件完全通用，杆长种类仅 2 种。运动学可达 D₀ 58–207 mm，补偿带 58–110，爬阶高度 149 mm。

#### 5.3.1 两态承力模型

| 状态 | 曲柄位姿 | 承力路径 | 舵机受力 |
|-----|---------|---------|---------|
| **驻留态**（轮行 / 站立，D₀=58） | 贴合承重面 | 轮→连杆→铰链→曲柄杆身→承重面→下甲板侧壁→机架（绕过舵机轴） | **≈0**（仅保持电流 ~50–80 mA/台） |
| **瞬态**（补偿 / 爬阶，D₀>58） | 离开承重面 | 轮→连杆→铰链→曲柄→舵机轴 | 0.51–1.92 Nm（§5.3.4） |

承重面为下甲板侧壁内侧匹配 D₀=58 时曲柄角度的斜凸台。D₀=58 时曲柄指向 α₁≈−209°（约水平上方 29°），凸台角度与之匹配。因构型完全对称，左右承重面几何一致。

载荷自紧负反馈：载荷经连杆产生拉力，其分量将曲柄压向承重面，越载越紧，为纯被动物理自锁，无需主动控制维持。瞬态动作结束后必须立即回 D₀=58 使曲柄重新贴合锁止，不可长时悬空。

#### 5.3.2 腿杆强度校核

杆件材料统一为 **9000he**（HP MJF PA12，粉末床熔融，成型密度 1.01 g/cm³），深边立放于 X-Z 面内。截面：大腿 13(深)×10(厚) mm，小腿 16(深)×10(厚) mm（分 a/b 两变体，Q 处 Y 向错层 4 mm）。轴向载荷为主，弯曲挠度非控制因素。

## 六、机构分析

分析对象为对称并联五杆髋关节单腿与整机轮行倒立摆。机构参数：d=52 mm，a=93 mm，b=149 mm，D₀∈[58,207] mm，轮半径 r=39.08 mm。单腿运动限定在 X-Z 平面。

方法：串联肘部解析解算 → 静力平衡法求扭矩 → 数值雅可比求可操作性 → 拉格朗日法构造 M/C/G → 爬阶轨迹动力学反解 → cart-pole 线性化 + LQR。

### 6.1 运动学

#### 6.1.1 机构解算

对称并联五杆，两髋点 A(−26,0)、B(26,0)，曲柄 a=93、连杆 b=149，输出点 Q=(0,−D₀)。每条链为「髋→膝(P)→末端(Q)」两连杆串联肘，由余弦定理解膝点位置：

```
cos α = (a² + |AQ|² − b²) / (2a·|AQ|)
α = atan2(Qy, Qx) ± acos(cos α)     # branch 决定肘朝向
```

左链取 `base−ang`（P₁ 落 Q 左侧），右链取 `base+ang`（P₂ 落 Q 右侧），形成对称非交叉构型，左右腿零件完全通用。

可达性条件 |a−b| ≤ |AQ| ≤ a+b，即 56 ≤ √(26²+D₀²) ≤ 242，代入得 D₀∈[49.4, 240.7]。机械限位 [58,207] 全段可达，余量充足。

#### 6.1.2 工作空间

Q 必须同时落在以 A、B 为心、内径 |a−b|=56、外径 a+b=242 的两个圆环交集内。距离约束仅为必要条件；真正可用的包络面还需满足非交叉分支约束——两膝 P₁、P₂ 分居 Q 两侧（P₁.x < Q.x < P₂.x），否则连杆交叉到非物理构型。

满足约束后，足端 Q 的可达工作空间为二维透镜形，面积 331 cm²。各高度 D₀ 的 X 方向可用带宽：

| D₀ | X 可达范围 | 宽度 | 半宽 |
|----|----------|------|------|
| 58（驻留） | [−45, +45] | 90 mm | ±45 mm |
| 70 | [−75, +75] | 150 mm | ±75 mm |
| 85 | [−93, +93] | 186 mm | ±93 mm |
| 100 | [−105, +105] | 210 mm | ±105 mm |
| 130 | [−117, +117] | 234 mm | ±117 mm |
| 145（峰值） | [−118.5, +118.5] | 237 mm | ±118 mm |
| 170 | [−116, +116] | 232 mm | ±116 mm |
| 207（上限） | [−98, +98] | 196 mm | ±98 mm |

驻留态补偿能力最小（±45 mm），因为 Q 紧贴圆环内孔边缘（|AQ|=63.5 ≈ 内径 56），X 偏移被内孔挤掉。中段（D₀=100–145）补偿能力最大（±105–118 mm），是左右地形差补偿的最佳工作区。需求值 ±25 mm 在驻留态即可满足。

对称步态（两曲柄镜像、Q 在 X=0 的 1-DOF 切片）用于纯垂向 D₀ 调节（驻留、爬阶）。两台舵机独立驱动（2-DOF）时 Q 可沿 X 偏移到 ±118 mm，实现左右腿独立的高低差与前后步态。

### 6.2 静力学

#### 6.2.1 瞬态关节扭矩

#### 6.2.2 静载保持扭矩

#### 6.2.3 力椭球与承载能力

核心关系 τ = Jᵀ·F（J 单位 mm/rad，τ 单位 N·mm = J[mm]·F[N]）。末端外载到关节扭矩的映射可视化为两组互补椭球。

**扭矩椭球**（A 图）：末端单位力 |F|=1 N 各方向 → 关节扭矩 τ₁、τ₂ 的椭圆轨迹。四个代表性 D₀ 的椭圆全部落在连续安全 1 Nm 圆内——任意方向 1 N 外载都不会让单关节超连续安全。实际两通道独立约束为 L∞ 方框（|τ₁|≤1 且 |τ₂|≤1），L2 与 L∞ 相差最多 √2≈1.4 倍，本文 L2 结论偏保守。

**末端力椭球**（B 图）：给定 |τ|₂≤1 Nm 连续安全，末端各方向能承受的最大外载包络：

| D₀ | 最差方向(N) | 最佳方向(N) | 堵转可承受(N) |
|----|-----------|----------|-------------|
| 58（驻留） | **41.8** | 43.1 | 122.8 |
| 100 | 13.4 | 17.6 | 39.5 |
| 150（峰值） | 11.0 | 11.3 | 32.4 |
| 207 | 8.7 | 14.7 | 25.5 |

驻留态 D₀=58 最差方向都能承受 41.8 N > 设计载 30 N，即使不走承重面、全悬空也仍在连续安全内——这是零力矩自锁的力雅可比证明：承重面是额外的降耗冗余，而非安全必要条件。D₀=150 峰值姿态最差方向仅能承受 11 N < 30 N，须靠堵转扭矩（可承受 32.4 N）短时支撑或回承重面锁止。

| 角速度峰值 | 3.2 rad/s |

动态增量 ~0.3 Nm 远小于静载保持 ~0.6–1.9 Nm，爬阶扭矩由静力主导，惯性可忽略——这是低质量桌面机器人的共性。角速度峰值 3.2 rad/s < 舵机空载 4.7 rad/s，速度可行（~30% 裕度）；若加速到 0.5 s/级则角速度 6.3 rad/s 超空载值，需降载或延长动作时间。

科氏/向心项 C·q̇ 用 dM/dt 一阶差分近似估算，量级 ~0.05 Nm，可忽略。0.5 s 快轨迹下此近似误差可能放大 2–3×，仿真中必要时改用解析 ∂M/∂q·q̇ 计算。

实际 30 mm 台阶爬升时 D₀ 仅需 58→~88（r39 + 台阶 30 + 间隙 ~19），行程仅 30 mm，加速度与扭矩相应更低，1 s/级的指标准则更宽松。

### 6.4 整机倒立摆与 LQR

驻留态腿被动自锁，整机降为经典轮式倒立摆：cart 质量为轮 mc=0.698 kg（DDSM315），pendulum 为机身及腿 mp=1.981 kg，摆长 Lp=56.0 mm（pendulum 质心相对轮轴：整机 COM 80.5mm 对应 pendulum 质心 95.1mm，减轮轴高 39.08mm），输入为地面驱动力 F=τ_wheel/R。

#### 6.4.1 线性化模型

绕不稳定平衡点 θ=0 展开，不稳定自然频率 ω_n=17.17 rad/s（周期 0.37 s），摔倒时间常数 ~65 ms，要求平衡环 ≥100 Hz。STM32 250 Hz 电机环满足此要求。

状态 [x, θ, ẋ, θ̇]，线性化 A/B 标准型，LQR 权重 Q=diag(10,50,1,1)、R=0.5。机身绕轮轴惯量以 mp·Lp²/3 近似（匀质杆假设），实际 240×196×50 mm 盒体分布质量与匀质杆差异可达 2–3 倍，LQR 增益对惯量敏感。当前 K 值按此近似计算，首件实测转动惯量（双线摆法）后重新标定。

#### 6.4.2 LQR 控制器

求解连续代数黎卡提方程：增益 K=[−4.47, −61.18, −5.82, −4.02]（对 [x,θ,ẋ,θ̇]，输入 F 单位 N），θ 项主导（−61.18），符合直觉。闭环极点 [−26.47, −11.93, −0.92±0.85j]，全部左半平面稳定。主极点 −0.92±0.85j 对应恢复时间常数 ~1.1 s，欠阻尼 ζ≈0.74，平衡收敛平滑。

#### 6.4.3 平衡扭矩校核

两表均为**每轮**扭矩（两轮共担，τ_single = F·R/2）：

| 倾角 θ | 准静态每轮 τ=Mg·sinθ·r/2 | %额定 | LQR 动态每轮 τ | %额定 |
|-------|---------------------|------|----------------|------|
| 5° | 0.045 Nm | 8% | 0.104 Nm | 19% |
| 10° | 0.089 Nm | 16% | 0.209 Nm | 38% |
| 20° | 0.176 Nm | 32% | 0.417 Nm | 76% |
| 30° | 0.257 Nm | 47% | 0.626 Nm | 114% |

准静态值为"静持不倒"的最小扭矩，30° 达 47% 额定。LQR 动态值为含 ẋ/θ̇ 阻尼项主动反馈的瞬时峰值：10° 内每轮 39% 额定；30° 大扰动超过单轮额定，靠两轮共担与电机短时过载（堵转 1.1 Nm = 额定 2×）吸收。整机增重后扭矩占用升高，直立临界裕度收窄至 0.19 kg，大倾角恢复能力下降。

轮行平衡的真正瓶颈在 RS485 ~250 Hz 控制环带宽与轮速（额定 0.82 m/s），扭矩在大扰动时亦趋紧张。

### 6.5 综合结论

| 维度 | 结论 | 数据 |
|------|------|------|
| 运动学 | 全程远离奇异（κ≤1.7），补偿带各向同性（κ≈1.03），γ≥31° | §6.1.4 |
| 工作空间 | 透镜包络 331 cm²；驻留补偿 ±45 mm，中段 ±118 mm | §6.1.2 |
| 静力学（设计载） | τ 0.51–1.92 Nm，D₀≤70 连续安全，峰值 1.92 < 堵转 2.94 Nm | §5.3.4 |
| 承载能力 | 驻留态可承 41.8 N > 设计载 30 N；D₀>70 进脉冲区 | §6.2.3 |
| 动力学 | 动态增量 ~0.3 Nm ≪ 静载，惯性可忽略；爬阶 1 s/级角速度可行 | §6.3.2 |
| 零力矩自锁 | 驻留 G≈0.05 Nm + 承重面绕过舵机轴 → 舵机近零受力 | §5.3.1, §6.3.1 |
| 轮行平衡 | 准静态扭矩裕度极大；LQR 闭环稳定；瓶颈在控制带宽与轮速 | §6.4 |

机构参数 d=52, a=93, b=149, D₀ 58–207 在运动学、静力学、动力学、整机平衡四个层面均自洽且裕度充足。所有瞬态动作遵循"脉冲→回承重面锁止"准则即可保证舵机长期可靠。

### 6.6 已知建模简化与局限

以下简化已在结论中采用，实际首件实测后应重标定，避免误用为精确值：

- **倒立摆忽略轮转动惯量与滚动约束**：LQR / cart-pole 把控制输入当作地面力 F 直接驱动质心，未建轮转角-平移约束 `x = −r·φ` 与轮惯量 `I_w`。部署侧以 `τ = F·R/2` 近似（两轮均分），对准直驱小惯量成立，但真实系统多一个快极点（轮动力学），高带宽下会偏离。
- **摆杆惯量取匀质杆近似**：`Ip = mp·Lp²/3` 为均匀杆绕端点惯量，与"质点平动项 `mp·Lp²` + 杆惯量"的拼接并非真实复合刚体惯量；应以 CAD 质量属性或实测惯量替换（见 §6.4 增益注明"首件实测后重标定"）。
- **cart-pole 降阶仅对腿锁止驻留态成立**：髋行程 ±1.52 rad 摆动时 COM 高度/惯量变化被忽略；65 ms 失稳时间常数仅针对驻留构型开环，RL 策略负责补偿摆动段。
- **roll 调平与高速 D₀ 门控为保守初值**：`ROLL_KP=0.15 mm/rad`、`D0_GATE_MAX_HIGH=120 mm`（"临界裕度 0.19 kg"）未从 roll/稳定性模型解析导出，需实测标定后依赖。
- **驻留态"零力矩"为结构论断**：静力求解器在所有载荷经连杆-舵机轴传递的假设下报 τ_dwell=0.51 Nm；实际承重面绕过舵机轴后舵机近零受力（≈0.05 Nm）。该分流未被求解器建模，故 0.51 Nm 是"承重面未接合"的最坏情况。
