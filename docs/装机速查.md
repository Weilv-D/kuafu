装机 / CAD 建模用的关键尺寸单一入口。详细信息按章节指针查阅三份核心文档。

## 一、五杆机构(蒙特卡洛 10 万采样验证)

机构四参数 (d/a/b/D₀) 经 `analysis/optimize/` 蒙特卡洛采样 + Pareto 前沿 + Dirichlet 权重扫掠验证。采用工程先验权重（扭矩/行程主导，传动角/条件数次要），选定解综合胜出率 50.5%、居候选第一。详见《结构设计》§二、§5.4。

| 符号 | 值 | 含义 |
|---|---|---|
| **d** | **52 mm** | 髋距，A(−26,0) B(26,0) |
| **a** | **93 mm** | 曲柄（大腿），左右对称，a/b=0.624 |
| **b** | **149 mm** | 连杆（小腿），左右对称 |
| **D₀** | **58–207 mm** | 足端下垂量（58 驻留 / 207 最大伸展） |
| τ_peak | 1.92 Nm | 瞬态峰值扭矩（<堵转 2.94） |
| γ_min | 31.9° | 传动角下限（≥30° 全程） |
| κ_max | 1.72 | 雅可比条件数峰值（<2 良态） |
| 行程 | 149 mm | 爬阶高度 |

> 半轮距 **Y=98 mm** 不参与机构优化，作为"滚转裕度 vs 整机宽度"的独立工程折中。

## 四、执行器（DDSM315 + ST3215）

### 轮毂电机 DDSM315（微雪）×2

| 项 | 值 |
|---|---|
| 轮径 / 胎宽 / 轴向最大宽 | Ø78.16（r=39.08）/ 34.8 / 39.81 mm |
| 安装 | 4× M2.1×6 @ Ø15.2 mm 节圆 |
| 重量 | 349±5 g（×2 = 698 g） |
| 扭矩 | 额定 0.55 Nm，堵转 1.1 Nm |
| 转速 | 空载 315±10 rpm，额定 200 rpm（轮缘 0.82 m/s） |
| 总线 | RS485，115200 固定，10 字节帧，CRC-8/MAXIM |
| 电压 | 额定 18V DC / 12–24V |

### 髋关节舵机 ST3215 C018（飞特）×4

| 项 | 值 |
|---|---|
| 机体尺寸 | 45.2（沿轴）× 35（X）× 24.7（Z）mm |
| 舵盘 | Ø19.95；4× M3 @ r14（Ø28 节圆） |
| 重量 | ~60 g（×4 = 240 g） |
| 扭矩 | 堵转 2.94 Nm @12V（1:345 金属齿）；连续安全 ~1 Nm |
| 转速 | 0.222 s/60°（45 rpm）@12V 空载；4.7 rad/s |
| 编码器 | 12 位磁编码器，0.088°/LSB |
| 总线 | TTL 半双工，串联，1 Mbps |
| 电压 | 6–12.6V |

### 安装接口

| 接口 | 规格 |
|---|---|
| DDSM315→叉口 | 4×M2.1 @ Ø15.2 节圆（预埋 M2 铜螺母，外径 Ø3.5） |
| ST3215→侧板 | 4×M3（Ø3.3 通孔）@ r14 + Ø20.5 中孔（过舵盘 Ø19.95） |

| Q 处错层 | 两连杆 Y 向垫片错层 4 mm |
