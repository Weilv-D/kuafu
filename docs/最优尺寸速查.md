# 夸父 KUAFU — 最优尺寸速查

本文档汇总经蒙特卡洛优化验证 + 实测定版的全部关键尺寸,作为装机 / CAD 建模的单一真源。
坐标系:原点在底盘几何中心,+X 前 / +Y 左 / +Z 上,Z=0 地面。

---

## 一、五杆机构(蒙特卡洛 10 万采样验证)

机构四参数 (d/a/b/D₀) 经 `analysis/optimize/` 蒙特卡洛采样 + Pareto 前沿 + Dirichlet 权重扫掠验证,选定解位于前沿密集包络内。

| 符号 | 值 | 含义 | 验证 |
|---|---|---|---|
| **d** | **52 mm** | 髋距(两舵机轴心距),A(−26,0) B(26,0) | — |
| **a** | **93 mm** | 曲柄(大腿),左右对称,a/b=0.624 | — |
| **b** | **149 mm** | 连杆(小腿),左右对称 | — |
| **D₀** | **58–207 mm** | 足端下垂量(58 驻留 / 207 最大伸展) | — |
| τ_peak | 1.92 Nm | 瞬态峰值扭矩(D₀=145) | <堵转 2.94 |
| γ_min | 31.9° | 传动角下限 | ≥30° 全程 |
| κ_max | 1.72 | 雅可比条件数峰值 | <2 良态 |
| 行程 | 149 mm | 爬阶高度(D₀_max−D₀_min) | — |
| Dirichlet 胜出率 | 24.5% | 200 组权重下综合胜出 | 居候选前列 |

> 半轮距 **Y=98 mm** 不参与机构优化,作为"滚转裕度 vs 整机宽度"的独立工程折中(落在边际收益递减拐点)。
